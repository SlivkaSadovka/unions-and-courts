# Профсоюзы и суды

Симуляция работы системы массового обслуживания жалоб работников через профсоюз и суды. Реализация покрывает требования варианта: диспетчер с возможностью прямого маршрута на суд, пакетная обработка заявок, календарь событий в пошаговом режиме и автоматический режим с итоговой статистикой.

## Установка

Для запуска требуется **Java 17**. Сборка через Gradle:

```bash
./gradlew build
```

Запуск в пошаговом режиме:
```bash
java -jar build/libs/app.jar --mode step --steps 20 --delta 0.5 --lambda 0.6 --buffer-size 8 --operators 3
```

Запуск в автоматическом режиме:
```bash
java -jar build/libs/app.jar --mode auto --duration 50 --delta 0.5 --lambda 0.6 --buffer-size 8 --operators 3 --direct-dispatch
```

Параметры
```bash
--mode — [step/auto]
--steps — количество шагов в пошаговом режиме
--duration — время моделирования в автоматическом режиме
--delta — шаг времени
--lambda — интенсивность потока заявок
--buffer-size — размер буфера
--operators — количество операторов
--direct-dispatch — включить прямую отправку заявок свободным операторам
```

## Расшифровка варианта

Источник (ИБ, ИЗ1): Работники, направляющие жалобы в профсоюз.
Поток бесконечный, распределён по закону Пуассона.

Диспетчер (ДП и ДВ): Профсоюзная организация. Диспетчер следит
за потоками жалоб и координирует их передачу в суды

Буфер (Д1ОЗ3): Архивы профсоюза. Накапливает и систематизирует
жалобы работников. Заявки обрабатываются по принципу LIFO
(последняя поступившая жалоба обрабатывается первой).

Прибор (ПЗ2): Суды различных инстанций. Время рассмотрения
жалоб распределено равномерно.
Дисциплина отказа (Д1ОО3): При переполнении очереди в
профсоюзе отклоняется самая старая жалоба.
Дисциплины постановки на обслуживание (Д2П1, Д2Б2):
1. Приоритет суда определяется его номером
2. LIFO: жалобы передаются в суд в порядке от последней
поступившей к первой
Заявка: Жалоба работника на нарушения трудовых прав, требующая
судебного рассмотрения.

Отображение результатов (ОР2, ОД1):
• Динамика: календарь событий (поступление жалобы, обработка
профсоюзом, передача в суд)
• Итоги: графики загрузки судов, количества отклоненных
жалоб, среднего времени обработки и рассмотрения
